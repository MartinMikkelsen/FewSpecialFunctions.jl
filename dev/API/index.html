FewSpecialFunctions.FermiDiracIntegralNorm — Function
FermiDiracIntegralNorm(j,x)

The Fermi-Dirac integral

\[ F_j(x) = \frac{1}{\Gamma(j+1)}\int_0^\infty \frac{t^j}{\exp(t-x)+1} \, dt\]

Returns the value $F_j(x)$
